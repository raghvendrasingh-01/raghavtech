# Raghvendra Singh

A modern 3D portfolio website built with React.js, TypeScript, and Three.js.

## Description

An interactive portfolio showcasing projects and skills with stunning 3D animations and effects.

## Technologies

- React.js
- TypeScript
- Three.js
- Tailwind CSS
- Vite

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
